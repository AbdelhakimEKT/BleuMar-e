## Convention de nommage

Convention retenue :

- toujours en minuscules
- toujours préfixé par `bleu-maree-`
- catégorie explicite : `hero`, `menu`, `people`, `logo`
- autres catégories utiles : `experience`, `desserts`, `details`
- description courte et stable
- pas d'espaces
- pas d'accents

Exemple :

- `bleu-maree-hero-dining-room-sunset.png`
- `bleu-maree-chef-portrait.jpg`
- `bleu-maree-detail-table-setting.jpg`
